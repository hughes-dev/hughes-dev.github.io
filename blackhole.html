Black Hole Gravity Simulator

A project using physics equations and code to simulate gravity.

This dynamic space simulation, rendered in 2D, showcases the gravitational interactions among space objects in a cosmic playground, featuring a multitude of celestial bodies each exerting gravitational forces on one another. The system is formulated around Newton's laws of gravitation, modeling the influence of each object's mass and distance on its gravitational pull.
The primary challenge of developing this application was overcoming the computational complexity that arises when every object is influencing every other object in the simulation. The initial design, where only a central black hole exerted a gravitational pull, was considerably less complicated than this more realistic approach. To overcome this, the code was carefully optimized to handle multiple interactions efficiently.
Another key challenge was the collision handling mechanism. Given the premise of the simulation, an object needed to increase its mass upon colliding with another. This led to challenges with managing object indices during the collision detection and resolution process. A solution was found by adjusting the indices appropriately when removing an object after a collision.
With these adjustments, the final product presents an intricate dance of celestial bodies, providing a fascinating demonstration of gravitational forces in action. This captivating 2D simulation provides a visually appealing and engaging representation of space phenomena, offering a glimpse into the cosmic ballet that unfolds in our universe.
